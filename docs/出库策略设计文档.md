# 出库策略遵循规则

本文档主要是关于出库策略的一些算法规则

出库工作站上位机在接收到出库任务后，需要通过出库策略模块，传入任务，得到出库计划，然后按照该计划开始执行出库任务

三台出库工作站共六个夹爪同时放货为一批次



1. 根据RCS下发出库任务后，库位上的货物信息以及出库任务中的包裹信息来安排出库计划

2. 输入会是QueueTask类型，以及库位信息

   ```python
   QueueTask：
       task_id = task.task_id				# 任务id
       task_types = task.task_types		# 任务类型 
       status = "pending"  				# 任务状态
       total_packages = task.packages_count # 包裹数量
       task = task							# 任务模型
       start_time = None					# 任务开始时间
       end_time = None						# 任务结束时间
   task:
       task_id: str                      
       task_types: str                   # 任务类型
       timestamp: str                    # 时间戳
       packages_count: int               # 包裹数量
       packages: List[Package] = field(default_factory = list)   # 包裹信息
   packages:
       package_id: str                      # 包裹ID
       box_type: str                        # 纸箱类型
       face_sheet: Optional[str] = None     # 面单信息
       logistics: Optional[str] = None       # 物流类型
       manual_process_type: Optional[str] = None  # 人工处理类型
       packaging_line: Optional[str] = None  # 打包线
       count: int                            # 包裹中货物数量
       goods: List[str] = field(default_factory = list)  # 货物信息
   ```

   ​    

3. 优先出任务当中货物数量只有一个，并且是去高速线1的包裹，然后出货物数量只有1个，去高速线2的包裹，接着出货物数量只有1个，去往赠品区，软包区的包裹，最后出货物数量大于1的包裹（数量依旧按照从小到大排序），如果有三个包裹的货物都是2，那尽量挑选都是前往同一条流水线的包裹，其他数量也是同理，尽量保证包裹前往的流水线一致

4. 取货尽量是从库位的低到高，从下到上

5. 如果连续三批的货物都是前往同一条流水线的的，则下一批先出一批去往其他流水线的货物（此条规则可以忽略数量少优先的规则），然后再继续出原来流水线的货物（但有多盒包裹正在出库时除外）

6. 单台工作站的两个夹爪同时往出库线上放货时候，放置的两个货物的目的地流水线要一致

7. 当有某个包裹的货物只在单台工作站上有的时候，其他两个工作站要停止出库，只有此工作站连续出库

8. 当有一个包裹的货物数量大于6也就是同时一批最多的出库数量时候，下一批必须继续出此包裹剩下的货物，并且只出此包裹的货物，不掺杂其他包裹（哪怕只有一个）

9. 如果出现有一个包裹只有第一个工作站的第一个夹爪，第二个工作站的第二个夹爪，第三个工作站的第一个夹爪能够抓取，则其余三个夹爪此批次不出货（其他情况同理）

10. 每个包裹必须连续出库，不能出现这个包裹有八个货物都在第一第二工作站上，这一批次第三工作站先去出其他的包裹，必须得等到第一第二工作站连续两批将此包裹出完后，第三工作站才能出其余的包裹

11. 对于多个工作站，尽量避免多个包裹交叉的情况出现

12. 跨工作站的多盒包裹尽量优先安排

13. 当前可以确保ABR放货时候，每个库位内的货物的sku都是一致的，所以在出库工作站当中每个库位内货物的sku也是如同在ABR车内一样都是一致的（有可能出现不一致情况，需要考虑）

14. 后排 3、4 位必须等待同层前排 1、2 位清空


优先级：
| 优先级 | 规则   |
| ------ | ------ |
| P1     | 规则7  |
| P1     | 规则8  |
| P1     | 规则9  |
| P1     | 规则10 |
| P1     | 规则6  |
| P2     | 规则12 |
| P2     | 规则11 |
| P2     | 规则5  |
| P3     | 规则3  |
| P4     | 规则4  |
